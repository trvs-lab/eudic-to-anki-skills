Overview

一个专注学习闭环的 Skill，不是脚本拼盘。

Unified Workflow

把导出、生成、校验、导入、清理连接成稳定流程，减少手动切换与中断。

Quality Gate

卡片在导入前自动经过格式与内容校验，保障批量导入后的可用性与一致性。

Built-in Assets

内置 TRVS-Lab 模板与工作流文档，开箱即可运行，不必从零搭基础设施。

Workflow

从输入新词到进入复习队列，路径清晰且可追踪。

01 · Data In

通过欧路 OpenAPI 按「昨天 / 日期区间 / 词表」拉取词条。

02 · Card Build

Agent 结合模板构建字段内容，确保卡片结构符合目标 Note Type。

03 · Ship to Anki

生成发音后使用 AnkiConnect 导入，成功后自动触发 Anki 同步。

What You Configure

只需要配置关键依赖，其他交给 Agent。

  • EUDIC_TOKEN：用于访问欧路词典 OpenAPI。
  • Anki Desktop + AnkiConnect：用于卡片写入与连通检查。

典型启动命令
